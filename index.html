3.设p = 23和a = 5，使用费尔马小定理计算a^{2020} mod p

因为$5^{2020} = 5^{2291+18}(mod23)$，
再由费马小定理得，原式=$5^{18}(mod23)$；
又因为$5^{2}(mod23)=2$，
再由同余关系可知，原式=$2^{9}(mod23)$，
又因为$2^{5}=32(mod23)=9$,$2^{4}=16$,
所以原式=916=144。

4.使用欧拉定理计算2^{100000} mod 55。

由定义4.1知，φ(55)=|{1,2,……,54}|=40
由欧拉定理得，$2^{100000} \equiv 2^{40*2500}(mod55)=1^{2500}(mod25)=1$。

5.手动计算7^{1000}的最后两个数位等于什么。

为得最后两个数位，等价于$7^{1000}(mod100)$
由定义4.1知，φ(100)=|{1,2,……,99}|=100-(50+20-10)=40
由欧拉定理得，$7^{1000} \equiv 7^{40*25}(mod25)= 1$,
所以最后两个数位是01。

1.Bezout定理的完整证明

构造集合 $S={am+bn:m、n∈Z 且 am+bn ≥0$,显然集合S非空，由良序原则，取其中最小值 $d = a r + b s d=ar+bs d=ar+bs$。
要证明d是a和b的公因子
即证$d|a,d|b$
利用反证法，假设d不能整除a，即$a = qd + r, 0 ≤ r < d$
因为$r = a - qd = a - q(ar+bs) = (1-qr)a - qsb ∈ S$
又因为$0 ≤ r < d$,与d是S的最小值矛盾，故r不存在，即$d|a$
同理得 $d|b$,
所以d是a和b的公因子
任取a和b的公因子d’，即$’|a,d’|b$
因为$d = ar + bs = q_1d’r + q_2d’s = (q_1r+q_2s)d’ = q_3d’$
所以$d’|d$,
可得d是a和b的最大公因子
所以$d = gcd(a,b) = ar + bs$
